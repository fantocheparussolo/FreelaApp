FreelaApp

Status do projeto:
- Em desenvolvimento

Tecnologias utilizadas:
- Java
- Java Swing
- MySQL
- JDBC
- NetBeans
- GitHub

Objetivo:
- Sistema desktop para cadastro de clientes e trabalhadores freelancers.

Funcionalidades:
- Cadastro de clientes
- Cadastro de trabalhadores
- Login
- Integração com banco de dados
- Seleção de tipo de usuário

Banco de dados:
- MySQL para armazenamento de dados.

Execução:
1. Abrir no NetBeans
2. Configurar banco
3. Executar projeto
